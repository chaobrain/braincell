# Legacy Archive

This directory stores archived examples, tests, and migration snapshots that are kept for reference only.

- `legacy/examples/`: old multi-compartment examples removed from the main examples tree.
- `legacy/reference/`: migration snapshots, including the former `develop/braincell` workspace.
- `legacy/tests/`: compatibility and legacy-only tests that are no longer part of the main test suite.

The active codebase now lives under `braincell/`, the active examples under `examples/`, and the main tests under `tests/`.
